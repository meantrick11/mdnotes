# Kadane 算法详解
## 一、算法定义
Kadane 算法（卡登算法）是用于解决**最大子数组和问题**的最优算法，核心目标：在一个包含正负整数的数组中，找到**连续元素组成的子数组**，使其元素之和最大。

算法时间复杂度为 $O(n)$（仅需遍历数组一次），空间复杂度为 $O(1)$（仅使用常数额外空间），是处理该问题的标准解法。

## 二、核心思想
**一句话总结：边走边算，只保留有用的历史，实时更新最大值。**

核心逻辑：
1. 维护两个关键变量：
   - `当前最大和`：以**当前元素结尾**的连续子数组的最大和；
   - `全局最大和`：遍历过程中记录的所有连续子数组的最大和。
2. 遍历每个元素时，做二选一决策：
   - 要么**重新开始**：以当前元素作为新子数组的起点；
   - 要么**延续历史**：将当前元素加入之前的连续子数组。
3. 永远选择和更大的方案，同时更新全局最大值。

## 三、核心公式
```
当前最大和 = max( 当前元素 , 当前最大和 + 当前元素 )
全局最大和 = max( 全局最大和 , 当前最大和 )
```

## 四、算法步骤
1. 初始化：`当前最大和` = `全局最大和` = 数组第一个元素；
2. 从数组第二个元素开始遍历；
3. 按核心公式更新`当前最大和`；
4. 用新的`当前最大和`更新`全局最大和`；
5. 遍历结束后，`全局最大和`即为答案。

## 五、示例演示
数组：`[-2, 1, -3, 4, -1, 2, 1, -5, 4]`
目标：找连续最大和（答案：`[4,-1,2,1]`，和为 6）

遍历过程：
1. 元素1：当前最大和=max(1, -2+1)=1 → 全局最大和=1
2. 元素-3：当前最大和=max(-3,1-3)=-2 → 全局最大和=1
3. 元素4：当前最大和=max(4,-2+4)=4 → 全局最大和=4
4. 元素-1：当前最大和=max(-1,4-1)=3 → 全局最大和=4
5. 元素2：当前最大和=max(2,3+2)=5 → 全局最大和=5
6. 元素1：当前最大和=max(1,5+1)=6 → 全局最大和=6
7. 后续元素：当前最大和均小于6，全局最大和保持6

最终结果：6

## 六、适用场景
1. 经典最大子数组和问题；
2. 股票买卖最佳时机（单次交易）；
3. 连续区间最大收益计算；
4. 动态规划基础应用。
