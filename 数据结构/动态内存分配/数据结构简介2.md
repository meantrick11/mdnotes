### 结构体

**结构体**：一个或多个变量的集合，这些变量可以是不同的类型(类似于创建了新的数据类型，与int、char等初始c语言类型等价)
结构体指针：`struct Student *p = &stu;`，结构体指针的引用值：`p->id`
**类型别名定义**：`typedef 数据类型 别名；`  ，例如：`typedef struct Student{...} Student;`  `typedef int double;`

eg：
```c
    struct Student{
        int id;
        char name[10];
        int age;
        double height;
        ...
    }

    //用typedef定义一个结构体，而省略结构体名
    typedef struct{
        int id;
        char name[10];
        int age;
        double height;
        ...
    }Student;

```

初始化调用：
```c
    struct Student stu = {1001,"zhangsan",20,1.75};
    //.属性修改结构体中的值
    stu.id=1002;
```

### 内存分类

**静态/全局变量**：静态声明的变量和全局使用的这部分内存，这些变量在程序开始运行时开始分配，直到程序才消失
**自动内存**(栈内存)：函数内部声明的便李艾那个使用这部分内存，在函数被调用时才创建
**动态内存**(堆内存)：根据编写要求动态分配内存，可以编写代码释放，内存中的内容直到释放才消失

>步骤：
>1. 申请内存：`malloc(sizeof(数据类型))`*返回的是void类型的指针*
>如果成功，返回堆内存上分配的内存指针
>如果失败，会返回空指针
>2. 使用内存
>3. 释放内存：`free(指针)`
