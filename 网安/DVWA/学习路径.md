# 推荐学习顺序（从SQL注入开始）
## SQL Injection (SQL注入)

核心：学习如何通过拼接Payload获取数据库数据，理解闭合、联合查询、报错注入等。

工具：手动尝试后，可引入sqlmap自动化验证。

## XSS (Cross Site Scripting)

顺序建议：DOM XSS → Reflected XSS → Stored XSS

核心：掌握不同反射/存储位置的XSS利用，重点理解DOM型中JavaScript对URL参数的解析。

## CSRF (Cross Site Request Forgery)

核心：学习如何构造恶意请求诱骗受害者执行操作，理解Token和Referer的防护原理。

## Command Injection (命令注入)

核心：学习利用管道符、分号等执行系统命令，掌握不同操作系统的命令差异。

## File Upload (文件上传)

核心：学习绕过前端JS验证、MIME类型检查、扩展名黑名单，最终上传WebShell。

## File Inclusion (文件包含)

核心：学习本地文件包含（LFI）和远程文件包含（RFI），结合php://filter等协议读取源码。

## JavaScript Attacks (JS攻击)

核心：前端逻辑完全由JavaScript控制，直接在浏览器中修改代码或变量绕过验证。

## CSP Bypass (内容安全策略绕过)

核心：学习CSP头如何限制资源加载，并在允许的策略下寻找绕过方法。

## Cryptography (加密)

核心：分析弱加密算法、不安全的随机数，对比Impossible等级中正确的加密实现。

其他 (Authorisation Bypass, Open HTTP Redirect, Weak Session IDs)

作为补充练习，巩固对会话管理、权限控制的理解。
